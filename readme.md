# About

A datapack library that handle the common behavior of tool's durability.

> If your custom tool behave differently than vanilla tool you will not want to use this datapack

## Usage

This library is designed to be *embedded* into a datapack and should not be downloaded directly.

Each function under `api/` directory contains documentation about its functionality and its parameters.

## Warning

Since this project is just an experimental project to test [Global Durability](https://mc-datapacks.github.io/en/conventions/global_durability.html) convention, it is not recommend to be used in a serious project and might contain critical bugs.
